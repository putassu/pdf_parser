\# PDF Visual Parser (Gemma 3 Vision)



Инструмент для глубокого структурного анализа PDF-документов с использованием Vision LLM (Gemma 3) и OCR-подсказок.



\## Особенности

\- \*\*Атомарный парсинг\*\*: Разделяет текст и графические сущности.

\- \*\*Pre-OCR Hints\*\*: Использует EasyOCR для навигации модели по документу.

\- \*\*Smart Compression\*\*: Адаптивное сжатие страниц для работы в рамках контекстного окна LLM.

\- \*\*Configurable\*\*: Все параметры (DPI, качество, эндпоинты) вынесены в `config.py`.
